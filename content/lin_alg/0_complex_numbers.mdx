# Introduction

Real numbers, ${\rm I\!R}$, are any numbers that make sense as a displacement. Ex. ($-2, \pi$).
Some expressions are meaningless; $2/0$, $\sqrt{-7}$

To express them a new variable/value is introduced: $i$.
The unit for an **Imaginary Number** can be defined as: $i = \sqrt{-1}$ and $\sqrt{-7} = 7i$

## Properties of the Imaginary Number

1. $i^2 = -1$
2. $i^3 = -i$
3. $i^4 = 1$
4. $i^5 = i$
5. repeat
