# Definition

a non-empty set equipped with:
1. a rule for adding any two vectors
2. a rule for scalar multiplication

such that the following axioms hold.

## Axioms

### Closure

- if $v_1, v_2 \in V$ then, $v_1 + v_2 \in V$
- if $u \in V, c \in \R$ then $cu \in V$
