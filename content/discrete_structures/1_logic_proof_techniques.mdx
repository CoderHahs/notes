# Definition

A proposition is a sentence that is either true or false (not both !) <br/>
We can combine propositions to obtain a new proposition. In order to do that, we can use *logical* connectives.

| Symbol   | Name        | Intuition      |
|----------|-------------|----------------|
| $\neg$   | negation    | not            |
| $\wedge  | conjunction | and            |
| $\vee$   | disjunction | or             |
| $\to$    | implication | if ... then    |
| $\iff$   | equivalence | if and only if |
| $\oplus$ | exclusive   | either         |
| $\forall$ | for all    | all have to be true |
| $\exists$ | there exists | at least one

### Negation of Predicates

- $\neg(\forall x P(x)) \equiv \exists x \neg P(x)$<br/>
- $\neg(\exists x P(x)) \equiv \forall x \neg P(x)$
